Uploader

Kind: global class
Extend: Component

new Uploader()

Param Type Default Description
[options.data] object = 绑定属性
[options.data.title] string => 按钮文字
[options.data.url] string => 上传路径
[options.data.dataType] string "json" => 数据类型。可以是：text、xml、json、script。
[options.data.data] object => 附加数据
[options.data.name] string "file" => 上传文件的name
[options.data.extensions] string | Array.<string> => 可上传的扩展名。默认为空，表示可上传任意文件类型的文件；可以为字符串，多个扩展名用,隔开，如：’png,jpg,gif’；也可以为数组，如：[‘png’, ‘jpg’, ‘gif’]。
[options.data.maxSize] string | number => 可上传的最大文件大小。默认为空，表示可上传任意大小的文件；如果为数字，则表示单位为字节；如果为字符串，可以添加以下单位：kB、MB、GB。
[options.data.disabled] boolean false => 是否禁用
[options.data.visible] boolean true => 是否显示
[options.data.class] string => 补充class
